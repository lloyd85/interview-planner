

## Installation


### E2E Notes
create bin folder in root directory and add chromedriver.exe, geckodriver.exe and selenium-server-standalone-x.x.x.jar


## Babel config file

```npm install babel-preset-env --save-dev```

```yarn babel-preset-env --dev```

Create `.babelrc` with
 ```
{
  "presets": ["env"]
}
```
